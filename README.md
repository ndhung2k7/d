# Chat Link Bot - Tự động rút gọn link

Chat bot tự động xử lý link với khả năng tải lên hastebin và rút gọn qua nhiều API.

## 🚀 Tính năng

- 💬 Giao diện chat giống Messenger
- 🔗 Tự động rút gọn link qua 3 API (anonlink, linkx, mual)
- ⚙️ Cấu hình linh hoạt (tiêu đề, định dạng, khóa API)
- 📜 Lưu lịch sử trò chuyện
- 🔒 Bảo mật với giới hạn tốc độ
- 🛡️ Chống kiểm tra phần tử cơ bản

### Yêu cầu

- npm hoặc yarn
